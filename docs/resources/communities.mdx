# Communities
- Comunidades são grupos de usuários, recurso necessário para que aja interação entre os mesmos, através dos praises

## New community

**ROUTE:** /community <br/>
**METHOD**: POST <br/>

<Accordion title="Dart Request Class">
    ```dart
        class CreateCommunityInput {
            CreateCommunityInput({
                this.id,
                required this.description,
                required this.ownerId,
                required this.title,
                this.planMemberLimit = 50,
            });
            String? id;
            String ownerId;
            String description;
            String title;
            int planMemberLimit;
        }
    ```
</Accordion>

<Accordion title="Dart Response Class">
    ```dart
    class CreateCommunityOutput {
        CreateCommunityOutput({
            required this.id,
            required this.description,
            required this.title,
            required this.members,
            required this.ownerId,
        });

        final String id;
        final String ownerId;
        final String description;
        final String title;
        final List<Map> members;
        }
    ```
</Accordion>

## Delete community

**ROUTE:** /community/:id <br/>
**METHOD**: DELETE <br/>

<Accordion title="Response">
    ```dart
    class DeleteCommunityOutput {
    DeleteCommunityOutput({required String communityId})
        : message = "Community $communityId deleted";
    final String message;
    }
    ```
</Accordion>

## Add members

**ROUTE:** /community/members <br/>
**METHOD**: POST <br/>

<Accordion title="Request">
    ```json
    {
        "idCommunity" : "uuid",
        "members" : [
            {
                "idMember" : "uuid",
                "role" : "member"
            }
        ]
    }
    ```
</Accordion>

<Accordion title="Response">
    ```dart
    class AddMembersOutput {
    String message = "Member(s) added successfully";
    }
    ```
</Accordion>

## Remove members

**ROUTE:** /community/members/remove <br/>
**METHOD**: PATCH <br/>

<Accordion title="Request">
    ```json

            "uuid2",
            "uuid3"
        ]
    }
    ```
</Accordion>

<Accordion title="Response">
