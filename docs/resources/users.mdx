# Users
- Os usuários da aplicação do ponto de vista do domínio da aplicação. Toda parte de autenticação é a realizada por outra aplicação

## New user

**ROUTE:** /user/signup <br/>
**METHOD**: POST <br/>

<Accordion title="Dart Request Class">
    ```dart
    CreateUserInput({
        required this.tag,
        required this.name,
        required this.email,
    });
    final String tag;
    final String name;
    final String email;
    ```
</Accordion>
<Accordion title = "Dart Response Class">
    ```dart
        class CreateUserOutput {
            CreateUserOutput({
                required this.tag,
                required this.name,
                required this.email,
                required this.id,
            });
            final String tag;
            final String name;
            final String email;
            final String id;
        }
    ```
</Accordion>
