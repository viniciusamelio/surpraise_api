# Praises
- Um praise simboliza um ato de valorização por uma ajuda ou uma ação feita por outra pessoa, que te impactou positivamente


## New praise

**ROUTE:** /praise <br/>
**METHOD**: POST <br/>

**Dart Class Request Body:** 
```dart
class PraiseInput {
  PraiseInput({
    required this.commmunityId,
    required this.message,
    required this.praisedId,
    required this.praiserId,
    required this.topic,
  });

  final String commmunityId;
  final String praiserId;
  final String praisedId;
  final String topic;
  final String message;
  late String id;
}
```

**Dart Class Response Body:** 
```dart
class PraiseOutput {
  final String message = "Praise sent successfully";
}
```

## Get praise by user

**ROUTE:** /praise/user/:id <br/>
**METHOD**: GET <br/>

**QUERYSTRING** :
```json
{
    "asPraiser" : true
}
```
```dart
    bool? asPraiser
```
- Allows you to query praises filtering as praised/praiser role. If null, it will return everything from that community
